## 📌 Project Overview

This repository documents my journey from a Python beginner (post-course) to a confident, job-ready Python developer. 

The goal is to build **rock-solid foundations** in Python through consistent problem-solving, code reviews, and iterative improvement — with the ultimate aim of landing a Python developer role (targeting fresh graduate / junior positions).

### Why This Journey?
- Completed a comprehensive Python course (13 chapters + 2 mini projects).
- Mega projects on hold until foundations are stronger.
- JavaScript background + limited LeetCode experience (16-20 problems).
- Need for **deliberate practice**, syntax mastery, and clean code habits.
- Long-term vision: Data Science, libraries (NumPy, Pandas, etc.), web development, and production-ready applications.

## 🗺️ Learning Roadmap

Problems are solved **one at a time** in a structured chat-based mentorship format:

1. **Easy Problems** → Syntax, control flow, data types, basic functions, string/list/dict manipulation.
2. **Medium Problems** → OOP, file handling, modules, error handling, algorithms.
3. **Hard Problems** → Advanced data structures, optimization, system design elements, real-world applications.

**Rules of Engagement:**
- One problem per session.
- Submit solution → Receive detailed code review (gaps, improvements, best practices).
- Iterate and improve before moving forward.
- Repetition of key concepts for retention.
- Mix of difficulty levels as needed.
- Problems inspired by LeetCode-style questions but customized for Python depth.
